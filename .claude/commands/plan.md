# /plan Command

Startet Aufgabenanalyse mit @architect.

## Verwendung

```
/plan <beschreibung>
```

## Ablauf

1. @architect führt TRIAGE durch
2. @architect erstellt PLAN
3. @architect wählt AGENTS
4. @architect definiert PIPELINE

## Danach

Agents in Reihenfolge aufrufen:

```
@ib-specialist <aufgabe>
@backend-architect <aufgabe>
@code-reviewer Prüfe Änderungen
```
